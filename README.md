# 脚手架项目

架构：http://www.bootcdn.cn/  (开源js库cdn)
前端MVVM框架：vue.js(2.4)
前端开发框架：bootstrap,vue-resource(1.3.4),jquery（3.2.1）
页面：html5+css+div

JSON工具：FastJson
数据库：h2
DAO：jpa（hibernate实现）
WEB框架：springboot（springMvc，spring）
代码简化：lombok
数据源：druid
缓存：guava
日志：logback
Web容器：jetty
测试工具：junit
SSO：shiro

http://localhost:8080/h2-console
